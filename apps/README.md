# apps

Cluster-specific resources that cannot be expressed as common `eecs-k8s`
Helm values are registered here as local Argo applications.

- `cilium-lb-ipam`: B LoadBalancer address pool and L2 announcement policy.
- `rook-ceph-poc-config`: B-specific Ceph POC post-configuration.
- `rook-ceph-rgw-poc`: fixed POC bucket claim and externally reachable RGW
  Service. The CephObjectStore itself remains selected through
  `patches/rook-ceph-cluster/values.yaml`.

Cluster-local app-of-apps definitions for b.

These apps are intentionally kept outside `eecs-k8s` when they are POC- or
site-specific. Tower Argo reads this directory through the `b-local-apps`
root Application generated by the common renderer.
